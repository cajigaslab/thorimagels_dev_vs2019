SciChart WPF SDK Examples Source Code
-------------------------------------

The full source code to the SciChart WPF Examples can be downloaded, cloned and built / run from our 
Github repostitory:

Find this at https://github.com/abtsoftware/scichart.wpf.examples 


To build the examples from source 
---------------------------------

Make sure that you follow the instructions at our Github repository above to setup the SciChart Nuget feed. 

You will need this for nightly builds of SciChart. 

Instructions can be found here: 
http://support.scichart.com/index.php?/Knowledgebase/Article/View/17232/37/getting-nightly-builds-with-nuget

After that just clone the repo and build it! 


NOTE: Each example source code can be exported 
----------------------------------------------

If you don't want to go to all the trouble of the above, you can export a single example to a stand alone
visual studio solution. Instructions can be found here: 

https://www.scichart.com/documentation/v5.x/webframe.html#Exporting_the_Source-Code.html
